# lichess_puzzle_parser

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.24. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

Parse lichess puzzles to a sql database table similar to what encroissant uses to store puzzles
